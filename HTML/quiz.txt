HTML 测验三
1.XML 和 HTML有什么区别？ 
    HTML 不要求标记的嵌套，配对等，不要求标记间具有一定的顺序，
    XML 则是严格要求嵌套，配对。
    HTML 不具备扩展性，而XML是原标记语言，可以用于定义新的标记语言，
    HTML 侧重于如何表现信息，而 XML 是侧重于如何结构化的描述信息。
2.HTML 标记的开始标记称为？ 
    HTML 标记的开始称为开始标签，通常是成对出现的，比如 <div> 和 </div>
3.使用哪个扩展名保存 HTML 文件？ 
    HTML 文件扩展名为 .html 或 .htm
4.用 Mac PC（苹果电脑）编写的 HTML 代码可以在安装了 windows 7 的 PC 上浏览，用户将能够看到与 MAC PC 上设计的相同的设计？ 
    正确
5.下列哪个是有效的颜色代码？  
    #000000 每一个 HTML 颜色代码包含有 “#” 符号以及 6 个字母和数字，这些数字都是十六进制的。比如 “FF” 在十六进制中代表十进制的数字 255
6.下列哪个选项用于增加行高？
    rowspan 属性规定单元格可横跨的行数
7.DOM 代表？
    DOM 是 Document Object Model 文档对象模型的缩写
8.DTD 是什么意思？
    DTD 是一种保证 html 文档格式正确的有效方法，可以通过比较 html 文档和 DTD 文件来看文档是否符合规范，元素和标签使用是否正确
9.链接标记中的 href 属性指定？
    标签的 href 属性用于指定超链接目标的 URL。
10.div 标签的作用是？
    div 标签可以把文档分割为独立的、不同的部分，现在大多数页面都使用的 DIV+CSS 布局

6.可以用 HTML 播放音频吗？  可以
7.下面哪个是 font 标签的属性？  face;size;color
8.当前文档和链接文档之间的关系是使用什么属性指定的？ html 的 link 标签是用于当前文档引用外部文档用的，rel 属性用于设置对象和链接目的间的关系
